Currently at Genuin Codebase, working on a Module Federation micro-frontend shell, a 60+ component design system, and embeddable SDKs (video, reel feed, AI chat) running on publishers like iHeart Media, TED, and Indian Express — collectively serving millions of page views a month.

Based in Ahmedabad. More at [devtejot.dev](https://devtejot.vercel.app/).

## Selected work

**[ApplyPilot](https://github.com/devtejot/ApplyPilot)** — An MV3 browser extension that takes an online job application from open to submitted in under two minutes. It detects ATS pages, pulls the job description, scores fit, drafts tailored answers, and fills the form. It deliberately stops short of submitting on its own. *(TypeScript)*

**[realtime-dashboard](https://github.com/devtejot/realtime-dashboard)** — Built to go deep on WebSockets: live typing indicators, message reactions, and file transfers, with the server kept stateless while still handling reconnects and dropped connections cleanly. Next.js + Socket.IO, deployed across Vercel and Render. *(TypeScript)*

**[shop-verse](https://github.com/devtejot/shop-verse)** — A Next.js commerce demo I built to push App Router patterns: mood-quiz personalization, price-history charts, product comparison, cart bundling, and a wishlist, with Zustand and TanStack Query holding state together. *(TypeScript, Tailwind)*
